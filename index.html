Matrix Calculator:

Matrices are representations of systems of linear equations, and matrix operations are at the core of everything starting from estimating traffic flows to Google's search algorithm.
I designed and built a matrix calculator using Python, which can compute a variety of operations including but not limited to:
• Adding, Subtracting, Multiplying, Scaling, Transposing and Exponentiating Matrices
• Converting matrices to their row-reduced echelon form (RREF)
I created a Matrix class which has methods to perform these operations. You can manipulate Matrix objects (which are instances of this class) in your python console using these methods.
